# 21 veranos

Invitacion digital mobile-first para un cumpleanos 21. Es una app estatica
con una portada, un menu principal por tabs, subpaginas de wishlist y una
pantalla final de confirmacion.

## Estructura

- `index.html`: portada / home.
- `html/menu.html`: menu principal con Detalles, Horario, Plan y Wishlist.
- `html/closing.html`: cierre / RSVP.
- `html/wishlist/accesorios.html`: opciones de accesorios.
- `html/wishlist/cosas-cute.html`: opciones de cosas cute.
- `html/wishlist/libros.html`: opciones de libros.
- `html/wishlist/ropa.html`: opciones de ropa.
- `css/styles.css`: paleta, layout, cards, navegacion y responsive.
- `js/app.js`: tabs del menu, estado de like, cuenta regresiva, modal RSVP e imagenes opcionales.
- `assets/images`: imagenes de portada, actividades y wishlist.
- `assets/icons`: iconos usados en la invitacion.
